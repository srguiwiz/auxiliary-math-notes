About

These Auxiliary Math Notes help keeping an engineering tradition – about what one guy used to know who knew how to make stuff and theory.

We are using this. It is auxiliary because there is more basic stuff you probably need to know, and there is more complex stuff. We are still writing it down as a memory aid and to exercise the mind. And it was relevant to the work that guy did.

Enjoy.
